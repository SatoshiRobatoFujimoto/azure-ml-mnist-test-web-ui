Azure Machine Learning - MNIST 確認用フォーム
=============


使い方、
PHPの動くWebサーバのドキュメントルートに

+ img/
+ js/
+ css/
+ detect.php
+ index.html
+ upload.php

を入れておく。

index.htmlにアクセスすると、
![AzureML MNIST 確認フォーム](https://github.com/wiki/tottok-ug/tottok-ug-azure-ml-mnist-test-web-ui/images/azureml-mnist-form.png)

のような画面が表示されます。
